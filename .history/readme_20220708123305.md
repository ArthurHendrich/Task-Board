# Task Board

<br>

- ReactJS
- NextJS
- Typescript
- CMS: Prismic
- OAuth: GitHub
- FireBase
- Paypal payment

## Desafio!!

- Projeto desenvolvido para um funcionamento de um task list

- Cadastrar usuario (GitHub Auth)
- Realizar pagamento de premium (Paypal)
- Criação de taskLists (deletar/editar/adicionar)
- Página para o apoiador
